## Prerequisites

- Node.js >= 20.x

## Getting Started

1. **Clone the repository**

   ```bash
   https://github.com/NRestFzn/platforms-auth-react-vite.git
   platforms-auth-react-vite
   ```

2. **Set up environment variables**

   ```bash
   cp .env.example .env
   ```

3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Start the app**

   ```bash
   npm run dev
   ```
